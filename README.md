# Digital Modulation Simulator

A Python-based simulator for visualizing digital modulation techniques in wireless communication systems.

## Supported Modulation Types

- BPSK
- QPSK
- 8-PSK
- 16-QAM
- 64-QAM

## Features

- Random bitstream generation
- Digital signal modulation
- AWGN channel simulation
- Adjustable SNR and bit length
- Constellation diagram visualization
- Signal waveform visualization
- Interactive Gradio interface

## Technologies

- Python
- NumPy
- Matplotlib
- Gradio
- Google Colab
